4-26更新了不少东西，其中包括被诟病已久的家族副本。
说说我目前对新家族副本的一些理解，以及对成员的一些要求。

-------
#### 变化
```
1. 名称更变为 深渊副本
2. 每日挑战次数变为3次
3. 每周有3个副本可以选择
4. 增加了终极副本
```

#### 改动影响

```
1. 更容易获得副本头像
2. 副本奖励更加直观
3. 缩小了家族里老鸟和新人的实力差距带来的影响
4. 每次的副本挑战更富有意义即附加值变大
```
-------

现在我更加详细的说说副本的改动。

首先我说一下之前家族副本我是怎么处理的：

原来的规则是每天两次，一周内副本达到一定的要求就能拿到奖励。我一般一周的前三天就能打满额定奖励，剩下来的四天我便不会去碰家族副本。不过我还是时刻留意右上角的家族进度，我早先就计算过，只要家族里超过90个玩家完成了每周的额定任务，家族总进度就能打满。

这里有几个问题不容忽视：

1.每周家族总进度需要90个玩家打满每周额定奖励，但现实是可能只有不到50人完成了每周额定奖励。

2.老鸟们一般3天就能打满一个每周额定奖励的量，但是每周的后四天就不碰副本了，为什么？因为奖励和投入不符。

3.鲜有人正真去计算过家族副本的数值即认真对待家族副本。

现在再说一下今天及以后的家族副本我是怎么处理的：

首先每天3次我要打满，因为每次都会有 **单独的奖励** 。副本的评级尽量打高，每次评级越高意味着我能得到 **更好的卡牌** 周日深渊副本中使用。

这里同样有几点要注意：

1.老鸟们愿意把次数打满，每次打完之后都能立即得到反馈。

2.因为每一次都有奖励，所以不存在一天不碰家族副本的情况。


至此，是我仅仅作为族员的考量。

往下是对副本于家族整体的分析。

-------

上文提及了，原来的家族副本机制，只要超过90个玩家打满每周额定，家族的每周奖励也能满，而老鸟们打满他们的额定只要3天，即6次副本挑战机会。记住这些数值。

接下来是现在的副本，有趣的的是这次更新相当于给老鸟们划了一次定义：
[![家族副本的数量要求](https://s2.ax1x.com/2019/05/29/VuhQW8.md.png)](https://imgchr.com/i/VuhQW8)

| 定位 | 标准|

|     新手     |     s以下       |

接下来以蝙蝠副本为例，一共需要99000个击杀

假定家族里有70个白秋（每天跑满3次且实力在新手）

99000÷323≈307 个挑战次数 

307÷210＜＜2 天

说明只要正常族员手不惨不到两天就能跑满一个副本。

再说其他两个

西瓜：66000÷220÷210＜＜2天

无尽：228000÷1321÷210≈2.4天

这三个副本，70个白秋打满甚至不需要一周，只需要6天足矣。

现实是，家族里新手以下的小白占多数，不可能有70个活跃玩家，大概只有20-30个。而对于这条公告，认真看到这里的 不超过双手之数。

-------
家族的发展需要每一个族员，当然我不会要求你们每天像上述白秋一样打满每天。但至少，希望你们知道自己可以为家族做的更多，并且家族发展起来，并不难。只需要你们的参与，以及一点点的付出。

这和小白更新的初衷 增加acu（平均同时在线人数）是一样的。
